# To-Do App

A simple to-do application for managing your daily tasks.

## Description

This project is a to-do list application that helps users organize and track their tasks efficiently.

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd to-do-app
```

2. Install dependencies (if any):

```bash
npm install
```

## Usage

Run the application:

```bash
npm start
```

## Script File Process

### Step-by-Step Implementation:

1. **Initialize the project structure**

    - Set up HTML, CSS, and JavaScript files
    - Create basic DOM elements

2. **Create task input functionality**

    - Add event listeners for form submission
    - Validate user input

3. **Implement add task feature**

    - Create task objects
    - Append tasks to the task list

4. **Add task completion toggle**

    - Implement checkbox functionality
    - Update task status

5. **Create delete task feature**

    - Add delete buttons to each task
    - Remove tasks from DOM and data structure

6. **Implement local storage**
    - Save tasks to browser's local storage
    - Load tasks on page refresh

## Features

-   Add new tasks
-   Mark tasks as complete/incomplete
-   Delete tasks
-   Filter tasks by status
-   Persistent storage using localStorage

## Technologies Used

-   HTML5
-   CSS3
-   JavaScript (ES6+)
